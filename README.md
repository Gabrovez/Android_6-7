## Лабораторная работа №6-7: Взаимодействие с пользовательским интерфейсом и State в Jetpack Compose. Написание автоматических тестов
### Название проекта

Tip Time — приложение-калькулятор для расчета суммы чаевых на основе введенного счета и процента. Приложение автоматически обновляет результат при вводе данных, поддерживает округление суммы и адаптируется под разные ориентации экрана.
### Цель работы
Изучение концепции состояния (State) в Android-приложениях и принципов работы декларативного UI фреймворка Jetpack Compose. Освоение паттерна State Hoisting, создание компонентов без состояния, работа с модификаторами, ввод данных и клавиатурой. Получение навыков написания локальных (Unit) и инструментальных (UI) автоматизированных тестов.
### Функциональность

Приложение реализует следующий функционал:

* Ввод суммы счета через текстовое поле с цифровой клавиатурой и иконкой.
* Ввод процента чаевых через отдельное текстовое поле с иконкой.
* Автоматический пересчет суммы чаевых в реальном времени при изменении входных данных.
* Переключатель (Switch) для включения/выключения округления суммы чаевых до целого числа в большую сторону.
* Поддержка навигации между полями ввода (кнопки Next и Done на клавиатуре).
* Адаптивный интерфейс с поддержкой вертикальной прокрутки для корректного отображения в альбомной ориентации.
* Локализация текстов через ресурсы strings.xml.
